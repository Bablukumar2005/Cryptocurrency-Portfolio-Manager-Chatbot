# Cryptocurrency Portfolio Manager Chatbot

## Overview
This project is a chatbot that helps users track and manage their cryptocurrency portfolio. 
It provides real-time cryptocurrency prices and portfolio insights.

## Features
- Real-time crypto price tracking
- Portfolio management
- Chatbot interface
- API integration with CoinGecko

## Technologies Used
- HTML
- CSS
- JavaScript
- CoinGecko API
- Gemini API

## How to Run
1. Clone the repository
2. Open index.html
3. Start using the chatbot
